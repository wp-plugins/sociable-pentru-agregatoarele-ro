Adaptarea pentru agregatoarele din .ro a pluginului Sociable. Incluse FTW.ro, problogger.ro si proddit.com


== Description ==

Pentru adaugarea de noi agregatoare, includerea altor seturi de imagini si suport general trimite un mail la paun.eugen [at] gmail.com. 

== Installation ==

Dezarhiveaza si uploadeaza in wp-content/uploads

Activeaza pluginul

Adminstrarea se afla in meniul Settings/Sociable .ro

== Frequently Asked Questions ==

Exista incompatibilitati cu alte pluginuri?

Nu, momentan nu au fost semnalate. Daca totusi ai de semnalat o astfel de problema foloseste mailul paun.eugen [at] gmail.com
